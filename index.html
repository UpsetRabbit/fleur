Fleur

A(n) (unmaintained) tiling window manager for Microsoft Windows that supports gaps, dynamic resizing, and much more...

Features

Designed around more than just keybindings, Fleur has several unique features

Click-drag-swap

Allows you to click a window and drag it over whichever slot you wish to swap its position with

Dynamic Re-tiling

As you resize a window, the tiling of all windows is automatically adjusted to accomodate

Multiple Workspaces

Enhance productivity by keeping separate virtual screens for different purposes and switch between them any time

Floating Windows

Floating and tiled windows on the same workspace, with the option to tile or untile a window at any time
